# Sales Performance Dashboard

## Project Overview
This dashboard provides an overview of sales performance using interactive charts and KPIs. It helps analyze revenue, profit, orders, and sales trends.
## Dashboard Preview
![image](Dashboard.png)


## Key Metrics
- Total Revenue
- Total Profit
- Total Orders
- Average Order Value

## Dashboard Features
- Revenue by Month
- Sales by Category
- Sales by Region
- Top Performing Products
- Interactive Filters

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Purpose
To help users monitor sales performance, identify trends, and support business decision-making.
